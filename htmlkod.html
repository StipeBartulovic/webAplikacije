Podnaslov: Primjena osnovnih HTML i CSS principa

Opis Projekta

Ovaj projekt je izrađen kao vježba za predmet vezan uz web dizajn. Cilj je prikazati strukturu stranice, formatiranje teksta, umetanje slika, tablica i linkova bez korištenja vanjskih biblioteka poput Bootstrapa.

Tekstualna Cjelina: Motivacija

Kroz ovaj projekt učimo kako HTML i CSS funkcionišu na osnovnoj razini. Razumijevanje strukture i rasporeda elemenata ključno je za svaku web stranicu.
